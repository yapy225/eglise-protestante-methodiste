# Église Protestante Méthodiste de Fontenay
## egliseprotestantemethodistefontenay.fr

### Fichiers du projet
- `index.html` — Site internet complet (1 fichier tout-en-un, prêt à publier)

### Automatisations intégrées
- **Écran de chargement** animé à l'ouverture du site
- **Bannière LIVE** — s'affiche automatiquement chaque dimanche de 14h à 16h
- **Badge "Message du lundi"** — apparaît automatiquement chaque lundi matin
- **Compte à rebours** vers le prochain culte (mis à jour chaque seconde)
- **Verset du jour** — change automatiquement chaque jour (7 versets)
- **Date pré-remplie** — formulaire de réservation pré-sélectionne le prochain dimanche
- **Menu mobile** responsive (hamburger)
- **Bouton scroll-to-top** (apparaît après 400px de défilement)
- **Bannière cookies** (s'affiche après 2 secondes, mémorisée)
- **Inscriptions sauvegardées** en localStorage (newsletter, formation)
- **Confirmation visuelle** après chaque formulaire soumis
- **Formulaire de notification** Niveau 3 (Formation Pastorale — en préparation)

### Pour mettre en ligne
1. Acheter le domaine `egliseprotestantemethodistefontenay.fr` (OVH, Gandi...)
2. Prendre un hébergement web (OVH, Ionos, o2switch...)
3. Uploader `index.html` à la racine du serveur
4. Configurer les emails : protestantmethodistefontenay@gmail.com
